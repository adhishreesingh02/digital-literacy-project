## Project Overview

This repository contains my Digital Literacy Portfolio developed as part of the CSE0001 course. Through this project, I have worked on different areas of digital literacy such as online safety, professional communication, basic coding, and cybercrime awareness.

This portfolio reflects my understanding of how to use digital tools in a responsible and meaningful way. It also shows how I can apply these skills in real-life situations as a student.



## Module Summaries

### Module 1: Digital Literacy Awareness

In this module, I learned about the basic concept of digital literacy and why it is important in today’s world. It mainly focused on safe internet usage, responsible online behavior, and how digital tools can support learning.

### Module 2: Digital Portfolio Development

This module focused on building a professional online presence using platforms like GitHub and LinkedIn. It helped me understand how maintaining a digital profile can be useful for both academic and career growth.

### Module 3: Coding & Collaboration Platforms

In this module, I worked on basic coding practice and also learned how to use collaboration tools. It included solving problems on platforms like HackerRank and creating Google Forms for collecting responses.

### Module 4: Email Etiquette & Communication

This module helped me understand how to communicate professionally in digital spaces. It covered how to write formal emails and maintain a proper tone and structure while interacting online.

### Module 5: Cybercrime Awareness

This module focused on understanding common cyber threats such as phishing and UPI fraud. It also included learning about preventive measures and how to stay safe while using online platforms.
