# Digital Literacy & Professional Skills Project

## Project Title  
Digital Literacy & Professional Skills Portfolio  

## Overview of the Project  
This project is based on the Digital Literacy course, where I completed different tasks to understand how digital tools and skills are used in real life. Through this project, I got to learn about things like creating awareness, using online platforms, solving basic coding problems, and understanding online safety.

Instead of making one single project, this includes multiple small tasks, and each one helped me learn something new. Overall, it gave me a better idea about how to use digital tools properly in academics and daily life.

---

## Features  
This project includes:

- A digital literacy infographic to spread awareness  
- A digital portfolio with profiles like GitHub and LinkedIn  
- A coding problem solved to improve logic  
- A Google Form created for a small quiz  
- Two professional emails for practice  
- A social media checklist  
- A cybercrime case study and prevention tips
